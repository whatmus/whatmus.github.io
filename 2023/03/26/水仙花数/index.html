水仙花数

发表于 2023-03-26 更新于 2023-03-27 分类于 编程 ， C# 本文字数：185 阅读次数：

水仙花数( Narcissistic number ) 是指一个 3 位数，它的每个位上的数字的 3次幂之和等于它本身。又被称为 自恋数 、 自幂数 和 超完全数字不变数 。
例如：$ 1^3+5^3+3^3=153 $
